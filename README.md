# Calculadora

Este projeto consiste em uma calculadora simples desenvolvida em C++. O programa permite realizar operações matemáticas básicas, como soma, subtração, multiplicação e divisão.

## Funcionalidades

Soma de dois números;
Subtração de dois números;
Multiplicação de dois números;
Divisão de dois números.

## Tecnologias Utilizadas

- C++
